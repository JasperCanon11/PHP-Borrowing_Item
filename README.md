# Laboratory Inventory and Borrowing Management System

## Overview
A web-based system developed using PHP and PHPMyAdmin for managing laboratory inventory and borrowing transactions. The system allows administrators to monitor stock levels, track borrowed items, and manage return status through a centralized dashboard.

## Features
- Admin authentication for secure access
- Inventory management (add, update, delete items)
- Stock monitoring and control
- Borrowing and return tracking system
- Record of borrower details and transaction history

## Technologies Used
- PHP
- PHPMyAdmin
- HTML
- CSS
- JavaScript
